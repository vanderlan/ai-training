# Taller AI Training Program

> 40 Hours Across 5 Modules (3 Weeks Part-Time)

This repository contains hands-on projects and exercises for Taller's internal AI training program. The training follows a practical, learn-by-doing approach with production patterns introduced from Module 1.

## 📚 Training Modules

### ✅ Module 1: GenAI Foundations & Vibe Coding
**Topics Covered:**
- How LLMs work (practical level)
- Model comparison: Claude vs GPT-4 vs Gemini
- AI-first development methodology
- Tools: Claude Code, Cursor, Copilot

**Project:** [URL Shortener](./module-1-genai-foundations/url-shortener)

---

### 🚀 Module 2: Advanced Prompting for Engineering
**Topics Covered:**
- Chain-of-Thought, few-shot, self-consistency
- Code-focused prompting (generation, review, debug)
- Multimodal: images, PDFs, screenshots
- Build personal prompt library

**Project:** [Code Analyzer Agent](./module-2-advanced-prompting/code-analyzer-agent)

---

### 🤖 Module 3: Agent Architectures
**Topics Covered:**
- Agent loop: Observe → Think → Act
- Memory systems and context management
- ReAct, Planning, Verification patterns
- Multi-agent orchestration

**Project:** [Migration Workflow System](./module-3-agent-architectures/migration-workflow-system)

---

### 📊 Module 4: RAG & Evaluation
**Topics Covered:**
- Embeddings, vector databases, chunking strategies
- Evaluation metrics: precision, recall, MRR
- LLM-as-judge evaluation
- Debugging and observability

**Project:** [RAG System with Evaluation](./module-4-rag-evaluation/rag-system)

---

### 🎯 Module 5: Production & Capstone
**Topics Covered:**
- Rate limiting, caching, security, cost optimization
- Responsible AI and governance
- Deploy, demo, showcase

**Capstone:** Choose from 4 projects:
- Code Review Bot
- Documenter
- Tech Debt Analyzer
- Multi-Agent System

---

## 📅 Schedule
**Typical Schedule:** 13-15 hours/week over 3 weeks (flexible, self-paced within cohort)

**Philosophy:** Learn by doing. Deploy with each module. Production patterns from module 1.

---

## 🛠️ Getting Started

1. Clone this repository
2. Navigate to the module you're working on
3. Follow the project README in each folder
4. Deploy and share your work!
